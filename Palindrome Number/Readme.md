# Palindrome Number

## Problem Statement

Given an integer `x`, return `true` *if `x` is a [palindrome](https://en.wikipedia.org/wiki/Palindrome), and `false` otherwise.*

## Example

**Input:**
```
x = 121
```
**Output:**
```
true
```
**Explanation:**
```
121 reads as 121 from left to right and from right to left.
```
